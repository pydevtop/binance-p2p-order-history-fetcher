# Descripción

**Binance P2P Order History Fetcher** es una aplicación profesional desarrollada en Python para merchants de Binance P2P, traders de criptomonedas, departamentos de contabilidad y empresas que necesitan exportar el historial completo de Binance P2P a Microsoft Excel.

La exportación estándar de Binance oculta parcialmente los nombres de los clientes mediante asteriscos (***), lo que dificulta la elaboración de informes contables, auditorías y documentación empresarial.

Este software permite generar informes completos de Binance P2P con nombres completos de clientes cuando dicha información está disponible en tu cuenta de Binance.
